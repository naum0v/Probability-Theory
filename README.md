## [Тема 1.](1.ipynb) Моделирование геометрической вероятности
Смоделируйте вычисление геометрической вероятности (на примере из лекции про метание дротиков в игровой круг, площадь которого в два раза меньше площади стены). Показать теоретический расчет и графическую иллюстрацию для рассматриваемого примера.

### Рассмотрим задачу:
Рассмотрим квадратную стену со стороной `side`. В центре стены (в точке `(x₀, y₀)`) расположена круглая мишень радиусом `R`. По условию, площадь стены в два раза больше площади круга.

Теоретический расчет:

Пусть `R` — радиус мишени.Тогда:

1. Площадь круга:  
   `S_круга = π × R²`

2. Площадь стены (по условию в 2 раза больше):  
   `S_стены = 2 × S_круга = 2πR²`

3. Сторона квадратной стены:  
   `side = √(S_стены) = √(2πR²) = R × √(2π)`

4. Теоретическая вероятность попадания в мишень при случайном броске в стену:  
   `P_теор = S_круга / S_стены = (πR²) / (2πR²) = 1/2 = 0.5`

В моделировании мы будем случайным образом бросать дротики в квадратную стену и считать, сколько из них попало в круглую мишень. Частота попаданий должна стремиться к 0.5 при увеличении числа бросков.
